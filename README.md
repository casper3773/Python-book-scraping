# Python-book-scraping

A Python script that scrapes weekly bestselling books from [Kitapyurdu](https://www.kitapyurdu.com) and saves them into a CSV file.

## Features
- Collects book titles and prices from Kitapyurdu’s weekly bestseller list
- Uses `requests` and `BeautifulSoup` for web scraping
- Exports results into a clean CSV file

## Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/casper3773/Python-book-scraping.git
cd Python-book-scraping
pip install -r requirements.txt

## Usage
Run the script:
python Python-book-scraping.py
